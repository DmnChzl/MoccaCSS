# Cursor

Below, utility classes to apply `cursor` property.

| Class           | Properties         |
| :-------------- | :----------------- |
| .cursor-auto    | `cursor: auto;`    |
| .cursor-default | `cursor: default;` |
| .cursor-pointer | `cursor: pointer;` |
