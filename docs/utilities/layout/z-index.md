# Z-Index

Below, utility classes to apply `z-index` property.

| Class  | Properties      |
| :----- | :-------------- |
| .z-0   | `z-index: 0;`   |
| .z-5   | `z-index: 5;`   |
| .z-10  | `z-index: 10;`  |
| .z-15  | `z-index: 15;`  |
| .z-20  | `z-index: 20;`  |

| .z-30  | `z-index: 30;`  |
